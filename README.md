# 🚀 1일 1알고리즘 & 코드 리뷰 SSAFY 알고리즘 스터디

> **"꾸준함이 실력을 만든다."** > 매일 알고리즘 문제를 풀고 서로의 코드를 공유하며 함께 성장하는 스터디입니다. 👋

<br>

## 📋 스터디 규칙

### 1. 1일 1알고리즘 & PR
- **방법**: 매일 문제를 풀고 해당 날짜 폴더 안에 본인 이름 폴더를 생성하여 코드를 업로드합니다.
- **파일**: 파일명은 상관 없으나 class명과 통일합니다.
- **규칙**: PR할 때 어떤 사이트의 몇번 문제를 풀었는지 포함합니다.
- **문제 선정**: 본인의 난이도에 맞게 문제를 설정하나 Solving Club이나 검정 기출 문제를 추천합니다.

### 2. 짝꿍 코드 리뷰 (Pair Code Review)
- **리뷰 대상**: 이번 주 배정된 리뷰 멤버의 전날 코드를 리뷰합니다.
- **마감 기한**: **다음날 오전 9:00 이전**까지 완료합니다.
- **리뷰 매너**: 비난은 금지! ❌ 성장을 돕는 건설적인 피드백과 칭찬을 주고받습니다.

### 3. 오프라인 스터디 준비
- 오프라인 모임 전까지 배정된 문제를 반드시 풀어옵니다.
- 본인의 코드를 동료들에게 명확하게 설명할 수 있도록 준비합니다. 🎤
